## Frontend Installation

### Node
#### Dependencies
- Node

#### Installation
- Go to the "frontend" directory in **modules/frontend**
- Install dependencies
```
npm install
```
- Build the application
```
npm run build
```
- Execute the application
```
npm run start
```
In case you want the application to reload on changes (hot reload)
```
npm run build:watch
```
